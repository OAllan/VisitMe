# VisitMe

## Clonar el proyecto

Usa el siguiente comando

```
git clone https://github.com/OAllan/VisitMe.git
```

## Instalar Pods

Instalar las dependencias del proyecto y generar el WorkSpace con el siguiente comando: 

```
pod install
```
